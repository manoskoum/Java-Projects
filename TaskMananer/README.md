# 📝 Task Manager - Java Console Application

A simple Task Manager application built in Java using Object-Oriented Programming principles.  
The project demonstrates CRUD operations, filtering, and basic state management through a console-based menu system.

## 🚀 Features

- Add new tasks
- Delete tasks by title
- Mark tasks as completed
- View all tasks
- View completed tasks
- View pending tasks
- Console-based user menu (switch-case)

- ## 🧠 Technologies Used

- Java
- OOP (Classes, Interfaces)
- ArrayList
- Scanner (Console input)

- ## 🏗️ Project Structure

- `Task` → Model class (represents a task)
- `TaskService` → Interface (defines operations)
- `TaskServiceImpl` → Business logic implementation
- `TaskManager` → Main class (user interface / menu system)

- ## 📌 Key Concepts Practiced

- Encapsulation
- Interfaces & Abstraction
- State management (completed / pending tasks)
- List filtering
- Basic user input handling
- Separation of concerns
